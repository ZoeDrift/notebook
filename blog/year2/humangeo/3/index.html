    第三章 认识论和分析技术¶

    Estimated time to read: 3 minutes

    认识论即主体对知识结构、知识的本质、知识来源、对知识的判断所持有的信念。

    第一节 经验主义和实证主义¶

    一、经验主义及其分析框架¶

    经验主义是与理性主义相对立的一种认识论，其秉持的观点是：感性经验是知识的唯一来源，一切知识都通过经验而获得，并且在经验中验证。约翰·洛克

    经验主义的方法论基础：归纳法

    • 特点：从事实到概念，从观察到总结，从局部到整体。根据全部事实确定规律性。
    • 缺点
      • 不清楚全部连续的推理，事实和假设之间就产生了逻辑上的缺陷
      • 普适性存疑
      • 常掺有归纳者的主观因素

    二、实证主义及其分析框架¶

    实证主义试图运用科学原理和科学方法解释社会现象。实证主义方法主张那些可重复观察到的现象是一种规律，而可重复的观察是由严格条件的。奥古斯特·孔德

    两种形式：

    • 基于证实的逻辑实证主义
      • 逻辑实证主义认为， 观察和逻辑 （或理性）是科学的两大支柱，前者是研究的核心，后者是人类思维可接受的世界描述。该学派发展出了用公式表达规律的特色。
      • 逻辑实证主义主要采用 演绎 的分析方法，强调通过科学方法的观察和研究，验证假设来发现知识。其过程为，基于已有理论和现实观察，提出假设；根据研究问题和假设，进行实验设计，收集大样本数据；利用科学分析方法对假设进行检验；如果数据支持假设，则证实了原有理论；如果数据不支持假设，就对理论进行修改，构建新的理论。
    • 基于证伪的批判理性主义
      • 批判理性主义是对逻辑实证主义的批判。它主张法则的正确性不取决于被证实的次数，而 取决于不被证伪 。在发现例外之前，法则是正确的。
      • 相对于提供大量的确定性证据，科学验证应该通过提供反例来进行。
      • 根据批判理性主义，没有任何一个理论可以完全被确证，因为可能存在着例外，只不过没被发现

    实证主义方法论对人文地理学的影响：

    • 发展了人文地理学的理论，开始寻求用空间分布的普遍规律解释事件。
    • 在空间科学的理论框架内更多地采用了演绎逻辑，从一般性规律出发，将其应用于特殊事件。其提出理论的过程和归纳法有很大的不同
    • 加强了人文地理学的科学化。采用的数字量化技术意味着空间分析上的精确性，取代了传统人文地理学模糊化的推论。

    缺点：

    • 将人看做理性 “经济人” ，并不考虑人的信念、价值、感觉等因素
    • 实证主义模型含有强烈的 普遍性 原理，将一个地方观察到的行为设想为所有行动的规范。
    • 地理系统是开放的 多变量系统 ，难于模拟实验，无法受控重演，因而演绎模型具有相对性。
    • 实证地理学是对空间的盲目崇拜，仅仅关注空间而 忽视社会社会环境和政治制度 的影响。

    第二节 人文主义¶

    一、人文主义地理学的产生¶

    1947发表的研究，强调地球表面的异质性和独特性，并非普遍性或共通性
    计量革命后批评机械性、抽象
    1976，段义孚正式使用人文主义

    • 研究主题包括，地方和景观的文化构建、日常生活地图的描绘和创造、地方与认同等
    • 强调 人的主观能动性 ，重视人类行为的丰富意义，而不仅仅是经济意义
    • 也强调意义、价值和解释
    • 对人口迁移的解释，不能仅仅看成事推力-拉力的共同作用的经济结果，也包括了情感依附、错位、流亡等情感的主观感受

    特点：

    • 我向性：人们对客体的认识基础都是来自认识主体“我”的感知-认知基础
    • 感情性：人们认识空间客体是包含感情的
    • 感悟性：感悟是基于经验且超越经验的人类思维

    人地理学的表达形式是主体反思式（感悟）的自述（我向）并以这种表达形式生成地理知识，或表达出叙述主体的地方感。

    批判：

    • 实证主义认为，人文主义地理学的研究方法过于主观，不够科学
    • 人文地理学过于强调人的主观能动性，却忽视了人类受制于社会、文化、制度等环境的制约

    第三节 结构主义¶

    一、 结构主义和结构主义地理学¶

    结构：结构是 关系 ，即多个要素在他们构成的系统中的功能关系。结构是由各个部分相互依存而构成的一个 整体 ，而部分只能在整体上才有意义

    结构主义方法：旨在探索表层的现象表达了哪些深层的关系（也就是结构）。根据诸因素之间的关系解释现实。

    结构主义地理学：是指以结构主义作为认识论的地理学，试图超越表面的地理现象，寻求以 深层结构 解释之。

    结构主义地理学与实证主义、人文主义地理学：

    • 实证主义 致力于发现秩序并总结其普遍规律，而 结构主义 地理学是理解事物的结构及其性质的地理学。
    • 人本主义 agency认为，地理现象是参与地理活动的个人行为的集合，对于地理现象的解释实际上是对个人行为的理解，并且任何宏观结构最终可以归因于个人行为。但 结构主义 structure明确反对这一观点，认为个人行为受制于行为发生所在的社会政治经济结构或社会结构。整体结构决定部分特征，离开整体结构就不能理解政治经济系统的各个组成部分，当然也不能理解个人的行为。

    结构主义地理学的特点：

    • 整体性：认识事物时，放在整体中分析
    • 共时性：深层结构的存在具有时段性，超过这个时段，不同的深层结构所决定的表层结构也就不同了

    批判：

    • 人本主义地理学认为，结构主义过于强调社会结构的影响，忽视了人的主观能动性
    • 结构主义地理学主要关注社会阶层的分层，忽视了性别，种族等其他社会分异

    二、马克思主义地理学的理论基础¶

    （一）唯物主义和辩证主义¶

    唯物主义的人地观：
    人堆世界的认识不能脱离客观的、外在的事实。客观的、外在的”地“包括第一自然（原生态的自然）和第二自然（被人类社会改变了的自然）

    辩证主义人地观：
    人与地之间的相互联系，彼此作用，不断循环演进。作为人地关系中的人，我们能做的是调整人类行为，认识自然、适应自然。

    作业

    找出实证主义指导下的学术论文，并分析说明为什么是属于该认识论的理由

    （二）劳动价值论¶

    市场决定的商品价值是商品的使用价值，由购买者和出售者共同决定；但是商品的劳动价值不是市场完全决定的；但是商品的劳动价值不是市场完全决定的，它由生产这些商品的社会必要劳动时间来决定。当工人在生产中所创造的劳动价值远大于他们真正得到的工资时，资本家便获得了剩余价值。

    （三）商品拜物论¶

    商品的性质
    购买者忽视商品“物美价廉”隐含着“血汗工厂”的剥削性质；劳动力也变成商品，人劳动目的就是为了在劳动力市场上交换获得工资，劳动者同自身的劳动活动相异化。

    消费的拜物教性质
    资本主义的商品生产改变了人类与非人类世界的关系，过度消费带来的浪费，引发了资源、环境等问题。

    （四）资本循环和资本积累¶

    包括货币资本循环、生产资本循环和商品资本循环

    资本主义缓解危机的途径：
    福特主义：通过大规模标准化专业化的垂直体系，寡头垄断市场结构来降低生产、控制市场供给；
    凯恩斯主义：通过国家干预政策，不断增加财政支出，以延续循环；
    后福特主义，：从生产标准化和大批量的商品，转向生产小规模、小批量产品的弹性专精模式，以满足快速变化的市场需求

    三、马克思主义地理学的研究¶

    （一）关于流动性¶

    • 所有促进信息、货物、资金、劳动力的流动的技术，在为人们提供流动福利的同时，也有助于生产资本的循环
    • 美国麦肯齐提出“时空压缩”概念：一格空间单元因为在其间的要素流动时间缩短，就好想被压缩小了一样。整个世界变成紧密联系、相互依赖的地球村
    • 马克思主义地理学者哈维用“时空压缩”解释了而今资本、劳动力和商品市场的扩大，延缓了发达国家和地区过剩造成的危机，实现他所定义的空间修复
    • 在全球经济竞争中出现了地方企业家主义，即地方政府以企业经营的形式来管理辖区，强化了时空压缩

    （二）关于空间生产¶

    空间生产：
    在一定历史时期城市的急速扩张、社会的普遍都市化的现象，可以分为城市空间生产、区域空间生产、全球空间生产。

    空间生产的本质：
    目的也是资本的积累，实现途径也是加速资本循环。
    绅士化过程（gentrification）（城市更新）：一种典型的资本主义空间生产，某些地段差到一定程度后，资本看到投资盈利空间后重新投入其中。

    （三）关于文化景观¶

    • 资本主义经济基础与上层建筑的配合
    • 空间本体论
      • 哈维将空间关系及其伴生的景观，看作是特定社会关系的生产和再生产，以此作为审视当今文化现象的新视角。这种范式被称为空间本体论。

    四、马克思主义地理学对后现代主义地理学的认识¶

    后现代主义的主张：
    认为没有绝对的规律或解释，只有多样的叙事和视角。后现代主义批判宏大叙事，反对使用一个统一、全面和普遍的视角来解释社会规律；而关注特定语境、群体或个体的局部、相对和多样的叙事，关注多样性和差异性。

    马克思主义对后现代地理学的批判：
    认为后现代主义认可短暂与分裂、接受异化和隔离、偏爱美学而非伦理学、迷恋结构主义，对资本主义政治经济运行视而不见。

    第四节 后现代主义地理学¶

    一、后现代主义地理学产生的背景¶

    两极分化的加剧、科技与自然的碰撞
    文化被纳入生产范畴，受市场供求关系的影响

    （1）政治经济背景的变化
    （2）后现代主义的萌芽
    首先被系统地用于20世纪60年代中期的建筑学领域。
    美国学者詹姆森认为后现代主义是对于整体的颠覆，定义为针对元叙事的怀疑态度。增强对差异的敏感，促进对不同事物的宽容能力。
    后现代主义是后现代社会（后工业社会、信息社会）的产物。
    （3）后现代地理学的产生
    当整个思想文化领域经历者颠覆性转折时，后现代主要对人文地理学的研究方式具有一定的特殊性，两者的相互认同始于前者对后者的研究特性（空间要素）的关注，而发展与后者对前者理论观念的借用和采纳。
    后现代主义地理学面临的事研究态度的转变。差异的忠实、经历的认同。

    二、后现代主义地理学的主要研究内容¶

    后现代主义反对空间科学和它的实证主义基础，强调破碎性、混乱性和怀疑性。对一些人文地理学者来说，后现代主义的吸引力在于它所强调的“特异、特殊及独特性”。为区域研究提供了一个多样化的视角

    • 多维空间和超空间
      • 地理空间、虚拟空间
    • 时空压缩
      • 决策的时间域缩短，生活风尚变换迅速，空间关系的激烈重组，空间障碍进一步消除，引发了强烈的时空压缩的感受。
    • 社会差异的文化研究
      • 社会分异关注社会群体与特殊群体之间的关系以及社会分类与不平等现象产生的原因。市场规律的作用是社会分异产生的原因，社会分异发哦之城市空间的分异。后现代主义关注差异性。

    第五节 时间地理学和行为地理学¶

    一、时间地理学¶

    有关人地关系的研究转向面向“人”的行为以及以“人”为中心的环境分析。

    时间地理学的理论体系围绕着连续时空环境下的个体活动研究而建立，其研究对象的内核是 人的活动 ，外围是与人的活动有关的时空背景，其间构筑了不同层次的概念模型，用以表达人的活动与外界环境之间的关系。如个体的时空路径表示的是在三维时空坐标系下个体的路径所形成的不间断轨迹。

    • 时空路径和时空棱柱的可视化表达与分析
      • 时空路径的可视化
      • 时空路径的解读
        • 个人路径不发生移动时与时间轴平行，发生移动时表示为斜线，斜率表示个体在空间中的运动速度
      • 驻点
      • 时空棱柱的表达与解读
        • 可画了个体在该时间范围内一定的时空预算下所有可能发生的路径的结合。
      • 时空路径与时空棱柱的差异
        • 时空路径记录过去和现在的活动轨迹，时空棱柱是面向未来的、基于现实基础的诸多路径的可能性
    • 时空行为的制约要素分析
      • 如何强调制约
        • 人的行为随意，应当认识那些围绕行为个体的制约条件，并尽可能阐明产生这些制约的主体
        • 注重“制约”分析，不仅关注那些可以观察到的外部行为，而且试图去分析那些没有发生的计划行为，以及行为发生后企图改善的期望行为
      • 制约的类别
        • 能力制约：人的生理构成及其所使用的工具而受到的个体行为的制约
        • 组合制约：个体之间或者个体与环境之间的交互需要满足一定的条件
        • 权威制约：指由于法律、习惯或者社会规范等对个体活动所产生的限制
    • 尺度分析
      • 个体层面：直观地揭示和比较不同群体的行为特征
      • 驻点层面：借助时空棱柱的形态可以分析活动的时间分配和空间分布，以了解城市节奏和活动系统
      • 社会结构层面：分析特定群体的时间供给和需求，了解如何通过将个体计划在时间和空间上进行分配来实现社会系统的能力
    • 时间地理学对规划和管理的意义
      • 城市规划和企业规划

    二、行为地理学¶

    产生的背景：
    1960计量革命推动下，地理学试图用数据模型来分析人文地理现象
    吸收心理学等学科知识探索人类空间行为决策
    重视个人态度、认知、偏好对其空间行为产生的影响
    在购物、迁居、住房市场、产业区位、农业、旅游业、休闲、归属感、犯罪、社区发展等诸多领域得到广泛应用

    • 空间认知分析
      • 认知：个体获取、加工、存储、转换、应用及交流信息的过程
      • 空间认知：是对空间知识的获取、加工、存储与处理
      • 空间知识：包括空间结构、空间实体和空间相互关系，是对空间和思想的内部化反映和重构
      • 空间认知设计个人活动与环境的关联，强调个体如何接受并利用外界信息以实现空间中的行为
      • 认知模式的研究是行为地理学研究的重要内容，其研究方法集中在认知地图
        • 认知地图：是空间认知的重要研究方法，指的是在大脑里形成的关于一个地方的空间结构与空间功能的地图
    • 行为决策与选择分析
      • 指从多种备选方案中选择一种方案的过程
      • 建立在既有的知识基础之上，还会收到一定的环境、社会、政治及文化背景的影响，感觉、情感、信仰及价值观也会影响决策，受到有意和无意的刺激
      • 决策过程包括两个刺激反应阶段
        • 第一阶段：决策过程的启动
        • 第二阶段：选择过程
    • 对规划管理的意义
      • 重视个人态度、空间认知、偏好对其空间行为产生的影响，将这些主观因素纳入行为决策模型中，弥补了早期的决策模型基于理性或者有限理性假设的不足，提高了对居民交通出行、迁居、购物等行为预测的准确性，也为城市规划和管理提供更为可靠的依据
      • 行为地理学的研究可应用于地震、洪水、疫情等公共突发事件的应急管理。认知地图可以展现公众对空间感知的特征，合理干预公共的风险感知，避免公众过渡恐惧或者过度放松
      • 可应用于旅游规划和管理
