    时空分析¶

    约 4081 个字 预计阅读时间 15 分钟

    时空分析 是指对同时具有 空间属性 和 时间属性 的数据进行分析，从中提取时空信息、发现时空规律和获取时空知识的方法。

    为什么要研究时空问题¶

    • 万物存在于空间和时间之中
    • 空间和时间是人类生存与生活的核心知识
    • 空间和时间信息是信息发展和应用的核心要素

    一些概念¶

    名词解释：时空分析

    时空概念¶

    • 什么是时空
      • “Spatio”指空间，“temporal”指时间。
      • “时空”（Spatio-temporal，或写作 spatial-temporal、spatiotemporal、space-time 等形式）这一术语用于数据分析，指代在空间和时间两个维度上收集的数据。它描述了在特定位置和时间的某种现象
    • 时空数据是指具有时空特征的数据。
    • 四种常见的时空（ST）数据类型：
      • 事件数据，包括在点位置和时间发生的离散事件（例如，城市中的犯罪事件发生率）；
      • 轨迹数据，指对移动物体运动轨迹的测量（例如，警用巡逻车的巡逻路线）；
      • 点参考数据，指在移动的时空参考点上对连续时空场进行的测量（例如，使用气象气球收集的地表温度测量）；
      • 栅格数据，指在时空网格的固定单元上对时空场进行的观测（例如，遥感数据）。

    时空数据的性质

    • “时间”的性质
      • 时间的概念化方式各不相同：
        • 线性（唯一的、有方向的时间段）
        • 循环（在特定时间范围后重复）
        • 其他
      • 时间是相对于某事物的
        • 时钟驱动时间 - 与特定时钟同步
        • 事件驱动时间 - 与某个事件同步，如某个经济事件（如股市崩盘）之前或之后；自然灾害（如台风、火灾、洪水等）之前或之后
        • 状态驱动时间 - 与状态变化同步，如河水水位高于警戒阈值，出生率低于\(2\%\)

    时空数据可以是：

    • 时间瞬间 - 特定于单个时间点，具有单个时间戳
    • 时间区间 - 在一个时间间隔内累积。具有开始和结束时间
    • 规则频率 - 时间范围是恒定的
      • 来自动物追踪设备的定期GPS定位
      • 十年一次的人口普查
    • 不规则频率 - 时间范围各不相同
      • 地震的发生
      • 政治边界的变更

    时空立方体¶

    时空立方体本质上是一种矢量数据结构，用采样点要素表示时间和空间。
    同质或异质的点用于表示特定时间断面或聚合特征。
    当多个时间断面组合在一起并符号化为一系列点要素的立方体时，就构建了时空立方体数据模型。

    时空立方体与条柱

    • 时空立方体结构具有行、列和时间步长。
    • 行和列决定立方体的空间范围，而时间步长决定时间范围。
    • 每个条柱在空间\((x,y)\)和时间\((t)\)上都有一个固定位置。覆盖相同\((x,y)\)区域的条柱共享相同的位置ID。包含相同时长的条柱共享相同的时间步长ID。
    • 无论点数据分布如何，立方体始终是矩形的，导致某些位置在所有时间步长上的点计数为零。对于许多分析，只有具有数据的位置（至少在一个时间步长上至少有一个大于1的点计数）才会被纳入分析。

    X、Y 轴 表示空间位置；
    T 轴 表示时间；
    每一个小单元叫 bin（条柱 / 时空单元）；
    每个 bin 对应“某个空间位置 + 某个时间段”内的统计值；
    在 ArcGIS 中，时空立方体通常保存为 netCDF 文件，即 .nc 文件。

    分析时空数据¶

    每个时间步单独分析¶

    核心思想：

    • 对每个时间步的数据分别分析；
    • 结果可以是每个时间步一个图层；
    • 或者合并成一个带时间属性的图层。

    使用时空约束进行分析¶

    核心思想：

    • 数据分析时同时考虑空间邻近和时间邻近；
    • 结果通常表现为一个图层。

    某些 GP 工具使用 spatial weights matrix，空间权重矩阵，通过 ST window 定义要素关系。

    ST window 包括：

    • critical distance，空间临界距离；
    • fixed interval in time，固定时间间隔。

    例如，判断一个犯罪事件的邻居时，不仅看它周围 500 米内的事件，还看它前后 7 天内的事件。

    • 工具
      • 热点分析 - 创建空间和时间上显著高值（热）或低值（冷）的聚类地图
      • 聚类和异常值分析 - 识别要素的相似性（高值或低值的空间聚类）或差异性（空间异常值）
      • 空间约束多元聚类 - 基于一组要素属性值和可选的聚类大小限制，找到空间上连续的要素聚类

    时间序列预测工具
    要求数据必须位于时空立方体中

    工具 描述
    曲线拟合预测 使用曲线拟合预测时空立方体每个位置的值。
    按位置评估预测 为时空立方体的每个位置选择多个预测结果中最准确的一个。这允许您将时间序列预测工具集中的多个工具与相同的时间序列数据一起使用，并为每个位置选择最佳预测。
    指数平滑预测 使用Holt-Winters指数平滑方法，通过将每个立方体位置的时间序列分解为季节性和趋势分量，来预测时空立方体每个位置的值。
    基于森林的预测 使用随机森林算法的改编版本，预测时空立方体每个位置的值。森林回归模型使用时空立方体上每个位置的时间窗口进行训练。

    真正的时空整体分析¶

    核心思想：

    • 所有时间步的数据同时参与分析；
    • 同时考虑空间维度和时间维度；
    • 结果可以是单个图层，也可以是新的时空立方体

    这一类方法主要对应 ArcGIS 的 Space-Time Pattern Mining Tools，时空模式挖掘工具。
    这些工具要求输入数据是 space-time cube，netCDF .nc 文件。

    新兴热点分析¶

    Note

    名词解释：新兴热点分析
    解释新兴时空热点分析，并举两个例子

    • 新兴热点分析
      • 新兴热点分析工具用于识别热点和冷点，并判断它们是否存在时间趋势（例如，新出现热点与持续热点或历史热点）。
      • 它通过查找在时空NetCDF立方体中的点密度（计数）或值聚类的趋势来实现
      • 该工具基于 space-time NetCDF cube时空NetCDF立方体，通常由以下工具创建：
      • Create Space Time Cube By Aggregating Points“通过聚合点创建时空立方体”
      • Create Space Time Cube From Defined Locations“从定义位置创建时空立方体”
      • Create Space Time Cube from Multidimensional Raster Layer“从多维栅格图层创建时空立方体”
      • 输入可以是：点密度；点计数；某个数值变量。
      • 统计方法
        1. 使用用户定义的空间关系概念；
        2. 对每个 条柱（bin） 计算 Getis-Ord Gi* 统计量；
        3. 每个 bin 得到：z-score；p-value；hot spot bin classification；
        4. 再使用 Mann-Kendall trend test 评估热点或冷点的时间趋势；
        5. 最后把每个位置分类为不同类型的新兴热点或冷点。
    • Mann-Kendall 趋势检验
      • Mann-Kendall统计量是对条柱计数或值及其时间序列的秩相关分析
      • 将第一个时间段的条柱值与第二个时间段的条柱值进行比较。
        • 如果第一个小于第二个，结果为+1。
        • 如果第一个大于第二个，结果为-1。
        • 如果两个值相等，结果为零。
      • 比较所有时间段对的结果被求和。预期总和为0，表示值随时间没有趋势；如果总和显著大于 0，说明上升趋势；如果总和显著小于 0，说明下降趋势。
      • 每个条柱时间序列的趋势记录为z得分和p值。
        • 小的p值表示趋势具有统计显著性。
        • 与z得分相关的符号决定趋势是条柱值增加（z得分为正）还是条柱值减少（z得分为负）。

    将每个研究区域位置分类如下：

    模式名称 定义
    未检测到模式 不属于下面定义的任何热点或冷点模式。
    新增热点 此位置在最后一个时间步长上是统计显著的热点，并且以前从未是统计显著的热点。
    连续热点 此位置在最后的时间步长间隔中，具有单一不间断的统计显著热点条柱序列。该位置在最后的热点序列之前从未是统计显著的热点，并且所有条柱中只有不到90%是统计显著的热点。
    加强的热点 此位置在包括最后一个时间步长在内的90%的时间步长间隔中是统计显著的热点。此外，每个时间步长中高值聚类的强度总体上在增加，并且这种增加具有统计显著性。
    持续的热点 此位置在90%的时间步长间隔中是统计显著的热点，并且没有明显趋势表明聚类强度随时间增加或减少。
    逐渐减少的热点 此位置在包括最后一个时间步长在内的90%的时间步长间隔中是统计显著的热点。此外，每个时间步长中聚类的强度总体上在减少，并且这种减少具有统计显著性。
    分散的热点 此位置是一个时有时无的热点。不到90%的时间步长间隔是统计显著的热点，并且没有一个时间步长间隔是统计显著的冷点。
    振荡的热点 最后一个时间步长间隔的统计显著热点，并且该位置在之前的时间步长中也有作为统计显著冷点的历史。不到90%的时间步长间隔是统计显著的热点。
    历史热点 最近的时间段不是热点，但至少90%的时间步长间隔是统计显著的热点。
    新增冷点 此位置在最后一个时间步长上是统计显著的冷点，并且以前从未是统计显著的冷点。
    连续冷点 此位置在最后的时间步长间隔中，具有单一不间断的统计显著冷点条柱序列。该位置在最后的冷点序列之前从未是统计显著的冷点，并且所有条柱中只有不到90%是统计显著的冷点。
    加强的冷点 此位置在包括最后一个时间步长在内的90%的时间步长间隔中是统计显著的冷点。此外，每个时间步长中低值聚类的强度总体上在增加，并且这种增加具有统计显著性。
    持续的冷点 此位置在90%的时间步长间隔中是统计显著的冷点，并且没有明显趋势表明计数的聚类强度随时间增加或减少。
    逐渐减少的冷点 此位置在包括最后一个时间步长在内的90%的时间步长间隔中是统计显著的冷点。此外，每个时间步长中低值聚类的强度总体上在减少，并且这种减少具有统计显著性。
    分散的冷点 此位置是一个时有时无的冷点。不到90%的时间步长间隔是统计显著的冷点，并且没有一个时间步长间隔是统计显著的热点。
    振荡的冷点 最后一个时间步长间隔的统计显著冷点，并且该位置在之前的时间步长中也有作为统计显著热点的历史。不到90%的时间步长间隔是统计显著的冷点。

    局部异常值分析¶

    局部异常值分析工具用于确定要素在整个数据集的时间范围内是属于聚类还是属于异常值。

    • 输入：时空NetCDF立方体
    • 输出：局部 Moran's I 指数、伪 \(p\) 值和表示聚类或异常值类别类型的类型代码（CO_TYPE）
    • 典型结果
      • 高值被高值包围：高-高聚类；
      • 低值被低值包围：低-低聚类；
      • 高值被低值包围：高值异常；
      • 低值被高值包围：低值异常。
    • 局部异常值分析和热点分析的区别：
      • 热点分析强调高值或低值是否聚集；
      • 局部异常值分析强调某个位置是否和邻居显著不同。
    • 应用
      • 某些地区是否有异常消费模式？
      • 是否某段时间疾病暴发率异常高？
      • 是否有郊区用水量明显高于邻近地区？
      • 最近一个月保险索赔数量是否突然上升？

    时间序列聚类¶

    时间序列聚类工具基于时间序列特征的相似性，将时空立方体中不同位置的时间序列划分为不同类别。

    简单来说，它回答的问题是：“哪些地方随时间变化的模式相似？”而不是只看“哪些地方数值高或低？”

    • 输入：时空NetCDF立方体
    • 输出：
      • 一个二维要素类，显示立方体中的每个位置，并用其聚类成员身份进行符号化，同时还会生成信息性消息。
      • 可选地，可以使用“输出图表表”和“启用时间序列弹出窗口”参数来创建显示每个聚类的代表性时间序列以及时空立方体每个位置的时间序列的图表。
    • 应用
      • 911 电话量分析：如果建立多年 911 报警电话的时空立方体，可以用时间序列聚类找出报警量变化相似的社区。
      • 零售店销售模式分析：大型零售商可以用该工具找出购买模式相似的门店。例如有些店圣诞季销售上升，圣诞后下降；有些店没有明显圣诞季效应。这些差异可以帮助预测需求和安排库存。
      • 气象温度波动分析：气象学家可以根据温度随时间波动的相似性，将不同地点分组。
      • 人口增长模式分析：人口学家可以用它比较不同国家的人口增长时间序列，识别相似发展模式。

    变化点检测¶

    检测时空立方体中每个位置的时间序列在什么时候发生统计性质改变。

    共享时空数据¶

    • map or layer package，地图或图层包；
    • exported images，导出图片序列；
    • video，视频；
    • time-enabled map layer for the web，Web 时间启用地图图层；
    • time-enabled image service for the web，Web 时间启用影像服务；
    • time-enabled map book，时间启用地图册。
