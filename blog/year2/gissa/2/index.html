    空间分析概念框架¶

    约 3135 个字 预计阅读时间 12 分钟

    基本概念¶

    Places 地点¶

    所有空间分析的核心都是“地点”的概念。

    地点通常有名称、坐标

    1884年的子午线会议将伦敦的格林威治天文台确立为经度基准。
    如今，1984年的世界大地测量系统（WGS84）及其随后的调整，为地球表面的每个位置提供了一对高精度的坐标

    Attributes 属性¶

    在GIS中，“属性”一词通常指与矢量地图中的单个要素或栅格中的像元相关联的数据表记录。

    属性分为定性的和定量的

    • 定量的Quantitative
      • 等距、等比、周期性的
    • 定性的Qualitative
      • 定类、定序

    Objects 对象¶

    点、线、面或体是属性的空间支撑

    • 在GIS中：
      • Point or multi-point 点或多点
      • Polyline or mult-polyline 折线或多折线
      • Polygon or multi-polygon 多边形或多多边形
      • Multipatch 多面体

    Maps地图¶

    历史上，地图一直是存储和交流空间数据的主要手段。

    • Real Maps 真实地图（实体地图）
    • Virtual/Digital Maps 虚拟/数字地图

    Multiple properties of places 地点的多重属性¶

    • 对象可能会重叠——河流可能会穿过树林，道路可能会跨过河流，而这三者都可能位于同一个区域内。
    • 图层与叠加：河流、道路、地形及它们的组合。

    Fields 场¶

    • 离散对象视图（Discrete-object view ）
      • 现实就像一个空桌面上散布着离散的、可数的对象，这些对象可以被分配到不同的类别中。
    • 现实是连续表面的集合，每个表面代表地球表面上某一种属性（如高程）的变化。

    Networks 网络¶

    线通常构成空间数据的对象。它们包括街道、道路、铁路和高速公路；河流、运河和溪流；以及社会学家研究的更为抽象的人际网络。

    网络构成了嵌入在二维或三维空间中的一维结构。

    Density 密度¶

    密度在离散对象和连续场概念化之间提供了有效的联系，因为密度表示单位面积上离散对象的数量，并且它本身就是一个连续场。

    在数学上，某种对象的密度是通过计算一个区域内此类对象的数量，并除以该区域的大小来计算的。

    Detail, Resolution and Scale 细节、分辨率与比例尺¶

    Note

    scale和resolution的关系，并举两个例子

    地球表面几乎是无限复杂的，看得越近，揭示的细节就越多。不可能建立一个包含每一个细节的表示模型，在现实中，必须决定包含多少细节。

    空间分辨率是一个阈值距离的术语，认为低于该距离的细节是不必要或不相关的。

    • 比例尺

      • 制图/地图比例尺：地图具有比例尺或代表性分数，即地图上的距离与现实世界中相应距离的比率。
      • 测量尺度或分辨率
      • 观测尺度：观测尺度的选择直接影响对地理现象的理解。
      • 操作尺度：指在政策制定和实施过程中应用空间分析结果的尺度。
      • 多尺度：全球尺度的问题可能涉及气候变化等现象，而局部尺度可能侧重于社区规划或城市内的土地利用。
        • 尺度跃迁：有时，现象会表现出尺度跃迁，这意味着一个尺度上的效应可能会对另一个尺度产生重大影响。

    • 多尺度与地图综合

      • 预选：决定包含哪些要素和排除哪些要素（例如仅保留主要道路，忽略小径等）。
      • 消除：移除太小/太短（低于某个阈值）的要素。
      • 简化：简化（平滑、拉直）线性或多边形边界，同时保留其基本形态。
      • 聚合：将不同的要素组合成一个更大的复合对象（例如将房屋群组合成多边形的“城市化区域”）。
      • 降维（折叠）：减少要素的维度，例如将城镇表示为一个点，或将宽阔的河流表示为一条线。
      • 典型化：通过用数量较少但在大致相同位置的同类对象替换多个对象，从而降低细节水平
      • 夸张：强调重要的要素，否则这些要素可能会被移除。
      • 分类与符号化：将相似的要素分组，并使用不同的符号系统来表示新的排列。
      • 冲突解决（位移）：识别并解决要素冲突（例如重叠、标注）。
      • 优化（修整）：改变要素的几何形状和对齐方式以提高美观度。

    拓扑结构 (Topology)¶

    在数学中，如果一个属性在空间的拉伸、缩放和扭曲中保持不变，则称该属性是拓扑的。

    维度：点、线、面和体之间的区别，它们的拓扑维度分别被称为 0、1、2 和 3。

    • 邻接性：包括地块、县和民族国家的接触/相邻。
    • 连通性：包括街道、道路、铁路和河流之间的交汇点。
    • 包含性：当一个点位于区域内部而不是外部时。

    拓扑是构建和编辑空间数据库以及空间分析中的一个重要概念。

    地理学定律¶

    核心问题: 地理现象在空间中是否存在普遍规律?

    地理学第一定律(空间相关性定律)¶

    任何事物都是与其他事物相关的，但相近的事物比遥远的事物关联更紧密。（越近越相关。）

    第一定律的应用：空间插值，空间聚类。

    地理学第二定律（空间异质性定律）¶

    Note

    名词解释：地理学第二定律 简答题：什么是空间异质性？举出三个分析方法和对应的应用场景

    空间异质性是指地理现象、变量关系或空间过程在不同地点表现出差异的现象。也就是说，同一个变量或同一种影响机制，并不会在整个研究区内保持完全一致，而是会随着空间位置的变化而变化。

    方法 作用 应用场景
    地理加权回归 GWR 分析变量关系是否随空间位置变化 研究房价影响因素在不同城区是否不同，例如交通、收入、学区对房价的局部影响
    局部空间自相关分析 LISA / Local Moran’s I 识别局部高值集聚、低值集聚和空间异常 分析疾病发病率、贫困率、犯罪率等是否在某些地区形成高-高或低-低集聚
    热点分析 Getis-Ord Gi* 判断高值或低值是否在局部区域显著聚集 识别交通事故热点、犯罪热点、疫情热点或商业消费热点

    空间的隔离，造成了地物之间的差异，即异质性(Spatial Heterogeneity)。

    应用案例：地理加权回归（GWR）、百度迁徙

    该定律强调地理现象的异质性，这意味着没有哪两个点或区域具有完全相同的地理特征。

    地理学第三定律¶

    地理环境越相似，地理特征越相近。

    应用案例：地理高斯过程回归

    地理学定律 核心思想 代表应用 模型核心目标
    第一定律
    （空间相近性）    		 地物相互关联，且距离越近，关联性越强。 空间自相关 检测和度量地理现象在空间上的依赖程度与聚集模式。
    第二定律
    （空间异质性）    		 空间数据关系存在异质性即非平稳性，在不同地方会变化。 地理加权回归
    （GWR）    		 刻画和建模变量关系随地理位置变化的非平稳性。
    第三定律
    （空间相似性）    		 地理环境/配置越相似，目标值越相似。 地理高斯过程回归
    （GGPR）    		 预测未知点的值，基于“相似的环境产生相似的结果”这一原则。

    空间关系¶

    核心问题: 地理对象之间如何相互联系?

    • 为什么要衡量空间关系
      • 空间分析基于空间对象位置和属性的表达，以及对象之间复杂空间关系的表达。 为了提高空间分析的能力，必须解决空间关系的描述和表达问题。
      • 空间关系可能由地理对象的几何特征（地理位置和空间现象的形状）引起，如距离、方向/方位、拓扑等。
      • 空间关系也可能由地理对象的几何和非几何特征引起，例如空间分布现象的统计相关性、空间自相关、空间相互作用、空间依赖性等。

    距离关系¶

    • 定性距离：近、远；近、较近、较远、远
    • 定量距离：欧几里得距离、曼哈顿距离、闵可夫斯基距离...
      • 欧几里得距离：\(d(A,B)=||A-B||=[\Sigma_{i=1}^{n}(a_{i}-b_{i})^{2}]^{1/2}\)

    方向关系¶

    • 方向：北、南、东、西，...
    • 测量方法：4方向模型、8方向模型、二维字符串模型、三角模型、最小外接矩形（MBR）

    拓扑关系¶

    Note

    名词解释：维度扩展九交集模型

    在数学上，拓扑关系是指在旋转、平移和比例缩放变换下保持不变的属性。
    在地理学上，它是指根据拓扑几何原理组织矢量空间数据的方式

    • 测量拓扑关系的模型包括：4I，9I 或 DE-9IM...
      • 四交模型
        • 它把每个空间对象分成两个部分：Interior，内部：对象里面的部分，记作 I(A)、I(B)；Boundary，边界：对象的边界线或边界点，记作 B(A)、B(B)。然后分别考察 A 和 B 的内部、边界之间是否有交集。
      • 九交模型
        • 将地理对象分为内部、边界、外部三部分(点只有内部和外部，没有边界)，通过判断两个对象的这三个区域之间是否有交点进而判断几何对象的拓扑关系。
      • 维度扩展九交集模型（DE-9IM）
        • 考虑两个对象的内部、边界和外部，它不仅判断交集是否为空，还判断交集的维度。DE-9IM描述的空间关系包括“相等”、“不相交”、“相交”、“接触”、“穿过”、“内部”、“包含”和“重叠”。

          这里的“交集的维度”是指：两个几何对象的某些部分相交后，交出来的结果是点、线、面，还是空集。

    模型 空间对象划分 矩阵规模 特点
    4I 模型 内部 + 边界 2×2 简单，能判断相离、接触、相交等基本关系
    9I 模型 内部 + 边界 + 外部 3×3 更完整，能区分更多空间关系
    DE-9IM 内部 + 边界 + 外部，并考虑交集维度 3×3 最精细，是 GIS 中常用的拓扑关系判断模型

    空间推理¶

    核心问题: 如何从空间数据中获得信息与知识?

    Note

    简答题：空间推理的概念和应用场景

    空间推理的概念¶

    空间推理是指基于空间数据中的空间关系、空间模式和空间过程，通过分析与推断获得新的地理信息或知识的过程

    空间推理现已广泛应用于GIS、机器人导航、高级视觉、自然语言理解、工程设计和物理位置的常识推理。

    空间推理的方法¶

    • 空间关系推理
      • 基于给定的空间关系模型，可以根据空间对象的位置推断空间对象之间的空间关系。
      • 基于空间对象之间已知的基本空间关系，可以推断空间对象之间未知的空间关系。
    • 不确定性（概率/模糊）推理
      • 代表性浓度路径（RCPs）是政府间气候变化专门委员会（IPCC）在气候变化模拟中使用的一组未来情景，用来描述温室气体浓度变化路径及其对地球辐射强迫的影响
      • 贝叶斯推理与空间推理，一个应用是最大似然估计。
    • 案例推理与空间推理
      • 基于常识的推理。
    • 时空推理
      • 在空间推理过程中加入时间因素。
    • 空间推理中的人工智能
