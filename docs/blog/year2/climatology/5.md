## 2.环流因子
### 大气环流
指地球大气层中具有稳定性的各种气流运行的综合表现，是地球内气流沿着稳定的路径进行不同规模运动的总称  

控制因素：地表辐射的不均匀分布，地球表面的摩擦作用、海陆分布和大地形等  

三圈环流模型：  
哈德利环流：赤道到副热带地区  
费雷尔环流：中纬度地区的环流  
极地环流：高纬度地区的环流

### 洋流

* 定义：又叫海流，是海水不同区域之间密度的差异造成的海洋
* 驱动力：海水密度差异、风、地球自转等
* 作用：海洋环流，与大气环流共同作用，传递和调节热量和盐分，对气候的形成有关键作用

夏季高原南坡和孟家湾上空是全球副热带最强加热带，其上南亚高压控制欧亚大陆，加热变化影响北半球

### 在青藏高原挖水汽通道能解西北干旱吗？  
>1970以前的故事？不行

>挖深的水汽通道行吗？不行

西边南风，东边北风，通道里面反气旋环流，通道里面没有热力差异没有动力

!!! note "一些gpt的解读"
    二、青藏高原的“热力作用”是什么？

    青藏高原夏季非常热（地面比周围平原高出几公里，还在烈日直晒下），  
    所以它的作用不仅仅是“挡风”，而是像一台“热力发动机”： 

    * 地面受热 → 空气上升（形成低压区）
    * 上升空气到高空后向外流出 → 高空形成反气旋（高压）
    * 整体气流：
    * 西侧（南亚一带） → 空气上升（降雨多）
    * 东侧（中国内陆） → 空气下沉（干燥少雨）

    也就是说，高原这个“大热源”让整个亚洲大气环流形成了一种**“上西下东”的垂直环流结构”**。

    三、那“反气旋环流”是什么意思？

    在北半球，高空高压区的风向是顺时针（反气旋）。

    也就是说：

    * 在高原上空（比如对流层上部）存在一个“顺时针转”的高压系统——南亚高压；
    * 这个环流决定了周围空气的流向：
    * 西侧 → 南风（带来湿气和上升运动）；
    * 东侧 → 北风（带来干冷空气和下沉运动）。

    四、那么“挖通道”在这样的环流下会怎样？

    想象一下：
    
    * 你在青藏高原上挖出一条“沟”，希望水汽能从南边的印度洋穿过去；
    * 但高原上空的空气环流已经是顺时针的反气旋结构，
    * 西边南风（往北输送湿气），
    * 东边北风（往南吹，带干空气），
    * 中间（通道那一带）其实是个辐散区或下沉区。

    👉 换句话说：空气在通道里不是向北钻，而是在下沉！

    五、为什么说“通道里面没有热力差异，没有动力”？

    空气要动起来（形成气流），需要“驱动力”：要么来自压力差（动力效应）；要么来自温度差（热力效应）。

    而青藏高原本身的热力差异是整个环流的核心：  
    高原热、周围冷 → 空气上升、环流形成。  

    但是如果你挖出一条“通道”，那里的地形就低了、热力减弱了，也就没有足够的热力驱动去吸引空气通过。

    于是：  
    没有温度差 → 没有上升运动；  
    没有上升 → 水汽无法抬升、凝结成雨；  
    反而通道附近会有下沉气流，空气更干。  

## 5.季风因子
现在被用来描述与热带辐合带年际维度震荡相关的大气环流和降水的季节性变化  

ITCZ  
定义：热带辐合带（ITCZ）是环绕地球赤道附近低层大气的一条由信风汇聚和上升气流组成的带状区域  
特征：ITCZ的上升气流导致云量增多和强降雨；ITCZ会随着太阳的季节性移动而南北移动  


### 全球季风
定义：  
地球上由于海陆热力差异和大气环流季节性变化引起的全球大规模风系和降水模式的季节性反转现象  
组成：
包括非洲季风、亚澳季风、美洲季风  

主要特征：  
1.风的季节性反转：1月和7月盛行风向和反向偏转至少大于120°  
2.降水的季节性集中：夏季降水量占全年总降水量的比例通常超过70%  

主要驱动机制：  
1.海陆热力差异：陆地和海洋的热容量不同，导致季节性温度差异的相对变化  
2.热带辐合带（ITCZ）的季节性移动：ITCZ的南北移动直接影响环流和降水的分布  

### 亚洲季风
亚洲季风=印度季风（南亚季风）+东亚季风  

**东亚季风**  
**东亚季风的特征**  
定义：影响东亚地区的一种显著的季节性风系变化  

形成机制：海陆热力差异（印度洋-太平洋-欧亚大陆），行星风系移动（ITCZ移动），青藏高原（热力和动力作用）  


**东亚夏季风**  

东亚夏季风：夏季（6-8月），影响东亚地区（包括中国东部、朝鲜半岛、日本及周边海域）的大尺度环流系统，其核心特征是暖湿气流从热带海洋向东亚大陆输送，导致显著的降水与高温天气  

主要特征：  

* 风向特征：
    * 低层盛行西南风或东南风，水汽输送
    * 高层受南亚高压（青藏高压）控制
* 降水特征
    * 中国东部：形成梅雨锋，降水集中且强度大
* 水汽来源
    * 南海、孟加拉湾、太平洋

* 梅雨：
    * 东亚地区在晚春到夏天期间特有的一个连续多雨的季节，梅雨期间空气湿度大，少有晴朗天气，是东亚季风最显著的特征   
    * 时间演变特征：5月初在南海和台湾地区开始，从5月到8月，随着雨带向北移动，会经历一系列的干湿交替阶段
    * 开始于南海5月，然后是长江流域和日本6月，最后是中国华北和日本7月，8月会迅速撤回至南中国

**东亚冬季风**  
是影响东亚地区冬季的主要大气环流系统之一，是由西伯利亚高压和太平洋阿留申低压之间的气压差驱动的，通常在每年11月-次年3月最为显著。冬季来自西伯利亚和蒙古高原的北方冷空气向南流动，影响东亚地区的气候现象。它主要表现为强烈的北风或西北风，带来寒冷干燥的天气。  

主要特征：  

* 风向特征
    * 冷空气的频繁爆发，从西伯利亚高压向南流动，带来急剧的降温。冬季风的风向主要是北风或西北风，风速通常较大，尤其是沿海地区
* 温度特征
    * 冷空气的南下导致东亚地区的气温显著下降，尤其是在中国东部、朝鲜半岛和日本
* 降水特征：
    * 北方来的冷空气与南方海洋暖湿气流的交汇也会带来降雪或降雨

**东亚季风的形成**

* 大气环流
* 海陆分布
    * 夏季观测，底层850毫巴高度场和风场，高层200毫巴高度场和风场
    * 冬季观测，对流层底层-西伯利亚高压、北风，对流层高层-东亚大槽、西风急流
* 青藏高原
    * 夏季热力-热泵效益
    * 冬季动力-机械强迫
