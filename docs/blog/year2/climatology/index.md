## 感受
这个课第一节课会要求分组，之后进行小组讨论、做小组作业。小组汇报展示的类型有三种：汇报、大作业开题报告、大作业答辩。其中汇报是老师给一个题目，小组内部自行分工进行汇报，大作业要求自己选题（最终的主题可以和开题的时候不一致）。  
j老师会登记每节课发言过和汇报过的同学，所以可以积极主动一点（笨人一次汇报三次发言）  
除了汇报，这学期进行了两次小测，一次是青藏高原专题（有提前通知），一次是气候诊断（无提前通知，上完就测），这两次都是不定项选择题，笨人考的稀烂。  
期末考在这学期相比历年卷新增了12个不定项选择，考得知识点很细。考场上写不定项写的很痛苦，考完也没有再试图回忆。不过虽然小测和期末都很难，但最后给分出人意料挺好的（）可能是我拼尽全力背诵ppt感动了上苍罢。
