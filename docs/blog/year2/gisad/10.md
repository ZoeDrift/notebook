## WebGIS后端（服务器端）概述

WebGIS后端的任务主要是对GIS数据进行管理，提供给WebGIS前端进行展示和分析。复杂的后端工作还包括对GIS数据的处理、渲染、分析等能力。  
GeoServer实际上就是一个完整的WebGIS后端，提供数据管理和渲染等能力。  

GDAL、OGR、GEOS、PROJ.4等工具库是地理空间数据处理与分析的基石，他们不仅应用于WebGIS，也被广泛的应用于桌面端、服务器端、移动端应用的后端服务中。

### Numpy和Pandas基础知识
- Numpy利用了Python的C语言底层接口实现了针对高维数组和矩阵的高效运算，极大地提升了数值计算效率，是Python应用于科学计算的经典第三方库之一。
- Pandas是Python语言的一个扩展程序库，用于分析结构化数据，Pandas以Numpy提供的高性能矩阵运算能力作为基础，可用于数据挖掘和数据分析，同时也提供数据清洗功能。

Numpy的核心对象是`np.array`  

```python
import numpy as np
a = np.array([1, 2, 3])
```

Numpy列表与Python列表区别：python列表可以混合类型，numpy数组通常要求元素类型相同   

numpy支持直接数学计算  
```python
a = np.array([1, 2, 3])
a * 2
# 结果：array([2, 4, 6])
```

从Series到DataFrame  
多个具有相同索引的Series合并在一起，形成类似表格的二维结构  

```python
import pandas as pd

df = pd.DataFrame({
    '姓名': ['张三', '李四', '王五'],
    '成绩': [90, 80, 70]
})
```

|   | 姓名 | 成绩 |
| - | -- | -- |
| 0 | 张三 | 90 |
| 1 | 李四 | 80 |
| 2 | 王五 | 70 |

**DataFrame数据访问**  

| 学号  | 姓名 | 成绩 |
| --- | -- | -- |
| 310 | 张三 | 90 |
| 311 | 李四 | 80 |

其中310、311是行索引，姓名、成绩是类名

- `df[列名]`的方式返回某一列的Series数据  
- `df[有关条件]`的方式返回符合“有关条件”的DataFrame切片   
- 由于DataFrame的行并没有行名一说，而是通过行索引来实现，因此不能通过行索引作为下标直接访问某一行的信息  

- 由于DataFrame的每一个单元格都可以通过“行索引+列名”和“行序号+列序号”精准定位，因此访问单元格也可以使用上述两种方法进行访问。  
- `.loc[行索引,列名]`方法支持“**行索引+列名**”进行访问。类似于列表切片，也可以使用冒号:表示访问某一区间（多行/多列）的内容。注意这里的括号是中括号，实际上相当于列表的切片
    - `df.loc[310, '成绩']`
    - `df.loc[310:312, '姓名':'成绩']`
- `.iloc[行序号,列序号]`方法支持“**行序号+列序号**”进行访问。类似于列表切片，也可以使用冒号:表示访问某一区间（多行/多列）的内容。注意这里的括号是中括号，实际上相当于列表的切片
    - `df.iloc[0, 1]`访问第0行第1列

**merge**  
```python
pd.merge(students, grades, on='学号')

pd.merge(left, right, left_on='student_id', right_on='id')
```

**join**  
join 更适合按照索引连接  
先把学号设置成索引`students = students.set_index('学号')`  
```python
students = students.set_index('学号')
grades = grades.set_index('学号')
```


| 类型    | 记忆方式    |
| ----- | ------- |
| left  | 以左表为准   |
| right | 以右表为准   |
| inner | 只要双方都有的 |
| outer | 双方所有都要  |

**分组统计 groupby**  

| 班级 | 姓名 | 成绩  |
| -- | -- | --- |
| 一班 | 张三 | 90  |
| 一班 | 李四 | 80  |
| 二班 | 王五 | 70  |
| 二班 | 赵六 | 100 |

按班级分组、取成绩这一列、计算每组平均值：  
```python
df.groupby('班级')['成绩'].mean()
```
groupby返回的是groupy对象，不是Series也不是DataFrame
