## 几何量算方法
GeoSeries和GeoDataframe的几何属性存储了面积、边界等基础信息：  

| GeoSeries 属性/方法       | 含义                             |
| --------------------- | ------------------------------ |
| `GeoSeries.area`      | 返回对象的面积，单位取决于对象的坐标系            |
| `GeoSeries.bounds`    | 返回对象的四至，类型为 `pandas.DataFrame` |
| `GeoSeries.length`    | 返回对象的周长                        |
| `GeoSeries.geom_type` | 返回对象的几何类型                      |
| `GeoSeries.x`         | 返回点类型的几何对象的 x 坐标               |
| `GeoSeries.y`         | 返回点类型的几何对象的 y 坐标               |
| `GeoSeries.z`         | 返回点类型的几何对象的 z 坐标               |


这些属性和方法能产生新的几何对象：  

| GeoSeries 属性/方法                             | 含义                                                    |
| ------------------------------------------- | ----------------------------------------------------- |
| `GeoSeries.boundary`                        | 返回对象的边界，降低一个维度的几何对象                                   |
| `GeoSeries.centroid`                        | 返回对象的中心点                                              |
| `GeoSeries.representative_point()`          | 返回对象的象征点，只保证点在对象内部，但是计算成本低                            |
| `GeoSeries.convex_hull`                     | 返回对象外包凸多边形；如果对象只有两个点，返回 `LineString`；只有一个点，返回 `Point` |
| `GeoSeries.envelope`                        | 返回对象的 aabb 外包矩形；如果对象只有一个点，返回 `Point`                  |
| `GeoSeries.buffer(distance, resolution=16)` | 返回缓冲区，`distance` 的单位由坐标系决定，`resolution` 决定转角的平滑程度     |
| `GeoSeries.simplify(tolerance)`             | 使用 Douglas-Peucker 算法，返回容忍范围内的简化几何对象                  |

**缓冲区：**
```python
GeoSeries.buffer(distance,resolution=16)
```
distance的单位由坐标系决定，resolution决定转角的平滑程度

**几何简化：**  
```python
GeoSeries.simplify(tolerance)
```
使用Douglas-Peucker算法，返回容忍范围内的简化的几何对象

**聚合**  

| 方法 / 属性                                        | 作用               |
| ---------------------------------------------- | ---------------- |
| `GeoSeries.unary_union`                        | 将多个几何对象合并为一个几何集合 |
| `GeoDataFrame.dissolve(by, aggfunc, as_index)` | 按字段聚合几何和属性       |


**分解**  

| 方法                                             | 作用                   |
| ---------------------------------------------- | -------------------- |
| `GeoSeries.explode(ignore_index, index_parts)` | 将 Multi 类型几何拆成多个单体几何 |

## 叠加分析
### overlay方法
| 方法 / 函数                                                                      | 作用                   |
| ---------------------------------------------------------------------------- | -------------------- |
| `GeoDataFrame.overlay(right_gdf, how, keep_geom_type=None, make_valid=True)` | GeoDataFrame 形式的空间叠加 |
| `GeoPandas.overlay(left_gdf, right_gdf, how, keep_geom_type, make_valid)`    | GeoPandas 函数形式的空间叠加  |

overlay 的 how 参数  

| 参数值                      | 作用                  |
| ------------------------ | ------------------- |
| `"union"`                | 并集                  |
| `"intersection"`         | 交集                  |
| `"difference"`           | 差集，保留 A 不与 B 重叠的部分  |
| `"symmetric_difference"` | 对称差集，保留不重叠部分        |
| `"identity"`             | 标识叠加，保留 A 并带入 B 的属性 |

### clip方法
| 方法 / 函数                   | 作用      |
| ------------------------- | ------- |
| `GeoDataFrame.clip(mask)` | 按给定范围裁切 |
| `gpd.clip(gdf, mask)`     | 函数形式的裁切 |

clip方法虽然看上去和overlay方法非常相似，但是overlay方法会独立地计算几何对象的每条记录。

## 空间链接和查询
| 方法 / 函数                                             | 作用        |
| --------------------------------------------------- | --------- |
| `DataFrame.join(right_df, how)`                     | 按索引进行属性连接 |
| `DataFrame.merge(right_df, how, left_on, right_on)` | 按字段进行属性连接 |
| `GeoDataFrame.sjoin(right_gdf, how, predicate)`     | 空间连接      |

```python
students.join(grades, how="left")
students.merge(grades, how="left",left_on="学号", right_on="学号")
gdf1.sjoin(gdf2, how="left", predicate="within")
```

连接的 how 参数

| 参数值       | 作用           |
| --------- | ------------ |
| `"left"`  | 保留左表全部记录     |
| `"right"` | 保留右表全部记录     |
| `"outer"` | 保留左右两表全部记录   |
| `"inner"` | 只保留两表都能匹配的记录 |
