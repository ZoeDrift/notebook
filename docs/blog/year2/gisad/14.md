# GIS空间分析—栅格数据
## 栅格数据
带有空间位置意义的二维数组。

- 栅格数据包含如下特征
    - 像元值
    - 像元大小
    - 像元深度
    - 栅格波段
    - 空间参照
- 应用
    - 遥感影像分析
    - 数字高程分析
    - 其他栅格数据
- 分析内容
    - 提取分析
    - 叠加分析
    - 距离分析
    - 邻域分析
    - 密度分析
    - 插值分析

### GDAL
GDAL，Geospatial Data Abstraction，是一个可读写大量的栅格空间数据格式的广泛应用的底层开源库。

特点 

- 使用C++编写，执行效率高、性能强。
- 支持通过Python、Java、C#等语言调用，在保留效率的同时提高了易用性。
- 支持GeoTiff (tiff)、Erdas Imagine Images(img)、ASCII DEM(dem) 等超过120种栅格数据格式。
- 被ArcGIS、Google Earth等诸多软件作为底层数据处理库使用。

### Rasterio
Rasterio是一个用于读写栅格数据集的Python库，它的设计目的是为了提供更高效、易用的方式来处理栅格数据，从而贯彻了Python的简洁性原则。

特点

- 简洁的 API 设计：Rasterio 提供了更直观和简洁的接口，使得栅格数据的读取、写入和处理更加容易。
- 更加 Pythonic 的使用体验：Rasterio 更符合 Python 编程的风格和惯例，它使用了标准的 Python 文件和上下文管理器，使得用户能够更自然地进行文件操作，支持类似常规文件读写的with语句，自动管理资源，减少内存泄漏的风险。
- 简化了空间参考和坐标转换：Rasterio 提供了更加直观的接口来处理坐标系统、投影和空间参考的转换。

## Rasterio方法
### 读取数据集
```python
rasterio.open("data.tif", "r")

# 配合with使用，自动打开和关闭文件
with rasterio.open("data.tif", "r") as dataset:
    ...
```

`"r"`:读取，`"w"`:写入  

### Dataset数据集属性/方法
| 属性 / 方法               | 基本写法                  | 作用          | 简要说明                  |
| --------------------- | --------------------- | ----------- | --------------------- |
| `dataset.meta`        | `dataset.meta`        | 查看所有元数据     | 返回字典，包含格式、大小、波段数、坐标系等 |
| `dataset.nodata`      | `dataset.nodata`      | 查看无效值       | 例如 `-9999` 表示无数据区域    |
| `dataset.width`       | `dataset.width`       | 查看宽度        | 即列数                   |
| `dataset.height`      | `dataset.height`      | 查看高度        | 即行数                   |
| `dataset.count`       | `dataset.count`       | 查看波段数       | 单波段、多波段影像都靠它判断        |
| `dataset.crs`         | `dataset.crs`         | 查看坐标系       | 决定栅格使用什么空间参考          |
| `dataset.transform`   | `dataset.transform`   | 查看仿射变换      | 决定像元行列号如何对应地理坐标       |
| `dataset.bounds`      | `dataset.bounds`      | 查看边界范围      | 左、下、右、上边界坐标           |
| `dataset.dtypes`      | `dataset.dtypes`      | 查看数据类型      | 例如 `uint8`、`float32`  |
| `dataset.tags(n)`     | `dataset.tags(1)`     | 查看第 n 波段元数据 | n 从 1 开始              |
| `dataset.colormap(n)` | `dataset.colormap(1)` | 查看第 n 波段颜色表 | 常用于分类栅格               |

### 读取波段
可以使用Numpy的数组切片功能或window参数读取部分内容  

| 函数 / 方法                         | 基本写法                                          | 作用        | 简要说明                         |
| ------------------------------- | --------------------------------------------- | --------- | ---------------------------- |
| `dataset.read(n)`               | `dataset.read(1)`                             | 读取第 n 个波段 | 返回 Numpy 二维数组                |
| `dataset.read()`                | `dataset.read()`                              | 读取所有波段    | 返回多维数组，形状通常是 `(波段数, 高度, 宽度)` |
| `dataset.read(..., window=...)` | `dataset.read(1, window=((0,1000),(0,1000)))` | 读取局部窗口    | 只读取指定行列范围，更省内存               |
| Numpy 切片                        | `dataset.read(1)[0:1000, 0:1000]`             | 提取局部区域    | 先读完整波段，再切片                   |

### 写入数据集
| 参数          | 示例                            | 作用     | 简要说明                |
| ----------- | ----------------------------- | ------ | ------------------- |
| `driver`    | `driver="GTiff"`              | 指定输出格式 | GeoTIFF 常写作 `GTiff` |
| `width`     | `width=dataset.width`         | 输出宽度   | 列数                  |
| `height`    | `height=dataset.height`       | 输出高度   | 行数                  |
| `count`     | `count=1`                     | 输出波段数  | 单波段写 1              |
| `dtype`     | `dtype="uint8"`               | 像元值类型  | 决定存储数据类型            |
| `crs`       | `crs=dataset.crs`             | 坐标系    | 决定空间参考              |
| `transform` | `transform=dataset.transform` | 地理变换   | 决定像元和坐标的对应关系        |
| `nodata`    | `nodata=-9999`                | 无效值    | 标记无数据区域             |

### Numpy统计方法
Rasterio 读出来的波段本质上是 Numpy 数组，所以可以直接用数组统计方法  

| 函数 / 方法          | 基本写法                  | 作用  | 简要说明        |
| ---------------- | --------------------- | --- | ----------- |
| `array.max()`    | `band1.max()`         | 最大值 | 例如 DEM 最高高程 |
| `array.min()`    | `band1.min()`         | 最小值 | 例如 DEM 最低高程 |
| `array.mean()`   | `band1.mean()`        | 平均值 | 例如平均高程、平均亮度 |
| `array.std()`    | `band1.std()`         | 标准差 | 表示数值离散程度    |
| `np.histogram()` | `np.histogram(band1)` | 直方图 | 统计像元值分布     |

### 地图代数相关方法 / 运算
| 类型           | 基本写法                            | 作用    | 简要说明          |
| ------------ | ------------------------------- | ----- | ------------- |
| 四则运算         | `band1 + band2`                 | 栅格相加  | 对应位置逐像元相加     |
| 四则运算         | `band1 - band2`                 | 栅格相减  | 常用于变化检测       |
| 四则运算         | `band1 * band2`                 | 栅格相乘  | 逐像元计算         |
| 四则运算         | `band1 / band2`                 | 栅格相除  | 常用于指数计算       |
| 比较运算         | `band1 > 100`                   | 条件筛选  | 返回布尔数组        |
| 布尔运算         | `(band1 > 100) & (band2 < 50)`  | 多条件筛选 | `&` 表示且       |
| 布尔运算         | `(band1 > 100) \| (band2 < 50)` | 多条件筛选 | `\|` 表示或      |
| 布尔运算         | `~mask`                         | 条件取反  | `~` 表示非       |
| `np.where()` | `np.where(band1 > 100, 1, 0)`   | 条件赋值  | 满足条件赋 1，否则赋 0 |

### 波段变换
| 方法     | 基本写法                    | 作用       | 简要说明                     |
| ------ | ----------------------- | -------- | ------------------------ |
| 读取所有波段 | `data = dataset.read()` | 得到多波段数组  | 形状通常是 `(波段数, 高度, 宽度)`    |
| 波段重排   | `data[[3, 2, 1], :, :]` | 重新组合波段   | 可用于假彩色合成                 |
| 单波段索引  | `data[0]`               | 获取第 1 波段 | Numpy 索引从 0 开始           |
| 单波段索引  | `data[3]`               | 获取第 4 波段 | Rasterio 编号和 Numpy 索引要区分 |

| Rasterio 波段编号     | Numpy 数组索引 |
| ----------------- | ---------- |
| `dataset.read(1)` | `data[0]`  |
| `dataset.read(2)` | `data[1]`  |
| `dataset.read(3)` | `data[2]`  |
| `dataset.read(4)` | `data[3]`  |


### 栅格镶嵌/拼接
| 函数 / 方法   | 基本写法                                     | 作用        | 简要说明         |
| --------- | ---------------------------------------- | --------- | ------------ |
| `merge()` | `mosaic, out_transform = merge(sources)` | 栅格拼接 / 镶嵌 | 把多个栅格合成一个大栅格 |


merge主要参数

| 参数                | 作用        | 简要说明                 |
| ----------------- | --------- | -------------------- |
| `source`          | 待合并栅格数组   | 传入多个打开的数据集           |
| `bounds`          | 输出边界      | 控制最终栅格范围             |
| `res`             | 输出分辨率     | 指定像元大小               |
| `use_highest_res` | 是否使用最高分辨率 | 多个栅格分辨率不同时使用         |
| `nodata`          | 无效值       | 指定输出 nodata          |
| `mask`            | 遮罩范围      | 控制输出区域               |
| `dtype`           | 输出数据类型    | 例如 `uint8`、`float32` |
| `indexes`         | 指定波段      | 选择要拼接的波段             |
| `resampling`      | 重采样方式     | 分辨率不一致时使用            |

### 重投影
| 函数 / 方法       | 基本写法                                          | 作用    | 简要说明               |
| ------------- | --------------------------------------------- | ----- | ------------------ |
| `reproject()` | `reproject(source=..., destination=..., ...)` | 栅格重投影 | 把栅格从一个坐标系转换到另一个坐标系 |

reproject() 主要参数  

| 参数              | 作用     | 简要说明            |
| --------------- | ------ | --------------- |
| `source`        | 原始栅格数组 | 被重投影的数据         |
| `destination`   | 目标栅格数组 | 存放重投影结果         |
| `src_transform` | 原始地理变换 | 原栅格 transform   |
| `src_crs`       | 原始坐标系  | 原栅格 CRS         |
| `dst_crs`       | 目标坐标系  | 新坐标系            |
| `dst_transform` | 目标地理变换 | 新栅格 transform   |
| `resampling`    | 重采样方法  | 重投影过程中需要重新计算像元值 |

### 栅格重采样

| 函数 / 方法                            | 基本写法                                    | 作用      | 简要说明       |
| ---------------------------------- | --------------------------------------- | ------- | ---------- |
| `dataset.read(..., out_shape=...)` | `dataset.read(1, out_shape=(500, 500))` | 改变输出形状  | 读取时直接重采样   |
| `resampling=...`                   | `resampling=Resampling.bilinear`        | 指定重采样方法 | 控制新像元值怎么计算 |


常见 Resampling 方法  

| 方法                        | 中文名   | 适合数据 | 简要说明      |
| ------------------------- | ----- | ---- | --------- |
| `Resampling.nearest`      | 最近邻   | 分类数据 | 不改变原始类别值  |
| `Resampling.bilinear`     | 双线性插值 | 连续数据 | 根据周围像元插值  |
| `Resampling.cubic`        | 三次卷积  | 连续数据 | 结果更平滑     |
| `Resampling.cubic_spline` | 三次样条  | 连续数据 | 平滑插值      |
| `Resampling.lanczos`      | 兰索斯   | 影像数据 | 图像质量较高    |
| `Resampling.average`      | 平均值   | 降分辨率 | 用平均值生成新像元 |
| `Resampling.mode`         | 众数    | 分类数据 | 用出现最多的类别  |
| `Resampling.gauss`        | 高斯    | 连续数据 | 有平滑效果     |


### 栅格裁剪
| 函数 / 方法  | 基本写法                                                          | 作用        | 简要说明            |
| -------- | ------------------------------------------------------------- | --------- | --------------- |
| `mask()` | `out_image, out_transform = mask(dataset, shapes, crop=True)` | 按矢量范围裁剪栅格 | 常用于按行政区、研究区裁剪影像 |


mask主要参数

| 参数            | 作用       | 简要说明               |
| ------------- | -------- | ------------------ |
| `dataset`     | 被裁剪的数据集  | 要裁剪的栅格             |
| `shapes`      | 裁剪范围     | 通常是矢量几何对象          |
| `all_touched` | 是否包含接触像元 | `True` 时范围偏大       |
| `invert`      | 是否反转范围   | `True` 表示保留外部      |
| `nodata`      | 无效值      | 裁剪外区域可设为 nodata    |
| `filled`      | 是否填充     | `True` 用 nodata 填充 |
| `pad`         | 是否边缘填充   | 控制边缘扩展             |
| `index`       | 指定波段     | 裁剪指定波段             |

### 矢量转栅
| 函数 / 方法       | 基本写法                                              | 作用    | 简要说明         |
| ------------- | ------------------------------------------------- | ----- | ------------ |
| `rasterize()` | `rasterize(shapes, out_shape=..., transform=...)` | 矢量转栅格 | 把点、线、面转成像元矩阵 |

rasterize主要参数

| 参数              | 作用       | 简要说明                         |
| --------------- | -------- | ---------------------------- |
| `shapes`        | 矢量对象和值   | 通常是 `(geometry, value)` 元组列表 |
| `out_shape`     | 输出栅格形状   | 例如 `(height, width)`         |
| `fill`          | 背景填充值    | 未被矢量覆盖的像元值                   |
| `nodata`        | 无效值      | 指定 nodata                    |
| `masked`        | 是否输出掩膜   | 只输出掩膜范围内容                    |
| `out`           | 输出对象     | 默认 Numpy 数组                  |
| `transform`     | 地理变换     | 决定输出栅格空间位置                   |
| `all_touched`   | 是否包含接触像元 | 和裁剪中的含义类似                    |
| `dtype`         | 输出数据类型   | 例如 `uint8`                   |
| `default_value` | 默认值      | shapes 只有 geometry 时使用       |
| `skip_invalid`  | 跳过无效几何   | 防止无效图形报错                     |

### 矢量转栅格
| 函数 / 方法    | 基本写法                                  | 作用    | 简要说明            |
| ---------- | ------------------------------------- | ----- | --------------- |
| `shapes()` | `shapes(source, transform=transform)` | 栅格转矢量 | 把相同值的像元区域转成几何对象 |


shape()主要参数  

| 参数             | 作用     | 简要说明         |
| -------------- | ------ | ------------ |
| `source`       | 被转换的栅格 | 可以是数组        |
| `mask`         | 掩膜范围   | 只转换指定区域      |
| `connectivity` | 连接方式   | 4 邻接或 8 邻接   |
| `transform`    | 地理变换   | 让输出矢量有正确空间位置 |

### 栅格矢量统计/分区统计
| 函数 / 方法         | 基本写法                          | 作用   | 简要说明          |
| --------------- | ----------------------------- | ---- | ------------- |
| `zonal_stats()` | `zonal_stats(vector, raster)` | 分区统计 | 统计每个矢量区域内的栅格值 |


zonal_stats() 主要参数  

| 参数            | 作用       | 简要说明                 |
| ------------- | -------- | -------------------- |
| `vector`      | 统计范围     | 矢量面数据，如行政区           |
| `raster`      | 被统计栅格    | 如 DEM、NDVI、降水栅格      |
| `stats`       | 指定统计量    | 如 `mean`、`min`、`max` |
| `all_touched` | 是否统计接触像元 | 控制边界像元是否算入           |
| `nodata`      | 无效值      | 排除无效像元               |
| `affine`      | 地理变换     | 等同于 transform        |
| `band`        | 指定波段     | 统计第几个波段              |
| `geojson_out` | 是否输出几何信息 | 返回 GeoJSON 形式结果      |
| `prefix`      | 字段名前缀    | 给统计结果字段加前缀           |
| `categorical` | 分类统计     | 统计每类像元数量             |
| `add_stats`   | 自定义统计量   | 添加自己的统计函数            |


### 总结
| 想做什么 | 用什么                          |
| ----- | ---------------------------- |
| 打开栅格  | `rasterio.open()`            |
| 读一个波段 | `dataset.read(1)`            |
| 读所有波段 | `dataset.read()`             |
| 写出栅格  | `dataset.write()`            |
| 看元数据  | `dataset.meta`               |
| 看坐标系  | `dataset.crs`                |
| 看地理变换 | `dataset.transform`          |
| 局部读取  | `window`                     |
| 数组统计  | `max()` / `min()` / `mean()` |
| 条件赋值  | `np.where()`                 |
| 波段组合  | Numpy 索引重排                   |
| 拼接栅格  | `merge()`                    |
| 重投影   | `reproject()`                |
| 重采样   | `out_shape + resampling`     |
| 裁剪栅格  | `mask()`                     |
| 矢量转栅格 | `rasterize()`                |
| 栅格转矢量 | `shapes()`                   |
| 分区统计  | `zonal_stats()`              |
