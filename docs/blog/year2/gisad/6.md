# Cesium三维地图框架入门
## Cesium特性
- 开源，基于 Apache 2.0 协议，可以免费用于商业和非商业用途。
- 用于创建具有高性能、高精度、高视觉质量和易用性的世界级3D地图框架
- 跨平台、跨浏览器
- 使用 WebGL 硬件加速

应用场景：  

- 支持2D,2.5D,3D 形式的地理（地图）数据展示
- 可以绘制各种几何图形、高亮区域，支持导入图片，甚至三维模型等多种数据可视化展示
- 可用于动态数据可视化并提供良好的触摸支持，支持绝大多数的浏览器和移动端浏览器
- 支持基于时间轴的动态流式数据展示

## 核心类
- `Viewer`: 展示三维要素内容的主要窗口，包含三维地球和一些基本控件
    - `Widgets` 组件: Viewer初始化界面上的组件，包括时间轴、位置搜索、底图选择等
    - `Scene`：构建场景的类，包含`primitives`（图元集合） `groundPrimitives`（贴地图元集合） 、`postProcessStages` 场景后期处理、相机、环境对象、事件等
    - `dataSources`: 矢量数据源集合
    - `mageryLayers`: 影像数据
    - `terrainProvider`: 地形数据
    - `Entities`: 几何实体集合
    - `Camera`：相机
    - `Event`：事件


terrainProvider:负责给 Cesium 提供地形  

```js
const viewer = new Cesium.Viewer('cesiumContainer', {
  terrainProvider: Cesium.createWorldTerrain()
})
```

dataSources:加载 GeoJSON、KML、CZML 等数据。  

```js
Cesium.GeoJsonDataSource.load('data.geojson').then(function(dataSource) {
  viewer.dataSources.add(dataSource)
})
```

Entity 是 Cesium 中非常常用的对象，可以表示地图上的一个具体对象  

```js
//添加一个点:在经度 120.1、纬度 30.3 的位置添加一个红色点
viewer.entities.add({
  position: Cesium.Cartesian3.fromDegrees(120.1, 30.3),
  point: {
    pixelSize: 10,
    color: Cesium.Color.RED
  }
})
```
### Viewer 初始化界面上的控件
- Geocoder
    - 位置搜索工具。
    - 可以搜索地址，搜索后相机会飞到对应位置。
    - 默认使用 Bing 地图搜索服务。
- HomeButton
    - 首页按钮。
    - 点击后，视图回到默认的全球视角。
- SceneModePicker
    - 视角模式选择器。
    - 可选择3D，2D，2.5D（哥伦布）视图之间切换。
- BaseLayerPicker
    - 底图选择器。
    - 可以切换底图和地形。
- NavigationHelpButton
    - 导航帮助按钮。
    - 点击后会显示鼠标如何缩放、平移、旋转地图。
- Animation
    - 动画控件。
    - 用于控制时间动画的播放。
-  CreditsDisplay
      -  版权信息显示。比如 Cesium、Bing、底图服务提供商等版权说明。
- Timeline
    - 时间轴。
    - 可以显示当前时间，也可以拖动时间。
    - 在动态轨迹、时序数据展示中比较有用。
- FullscreenButton
    - 全屏按钮。
    - 可以让地图进入或退出全屏状态。

### 取消相应控件的创建
关闭默认控件：  
```js
const viewer = new Cesium.Viewer('cesiumContainer', {
    animation: false,
    timeline: false,
    geocoder: false,
    homeButton: false,
    sceneModePicker: false,
    baseLayerPicker: false,
    navigationHelpButton: false,
    fullscreenButton: false
})
```

关闭版权信息：  
```js
viewer._cesiumWidget._creditContainer.style.display = 'none'
```
或者在css写  
```css
.cesium-widget-credits {
  display: none !important;
}
```

## Cesium空间坐标系统
地理坐标系：基于WGS84椭球体模型，使用经度、纬度和海拔高度表示地球表面点的位置的坐标系。 Cesium中的地理坐标单位默认是**弧度制**，Cartographic类用于定义弧度地理坐标。  

经度和维度单位为弧度，高度单位为米  
```js
var cartographic = new Cesium.Cartographic(1.57,2,200);
```

**笛卡尔空间直角坐标系**   
三维空间直角坐标系，原点在椭球的中心，空间直角坐标可以看作以椭球中心为原点的空间直角坐标系中的一个点的坐标。Cesium中用`Cartesian3`类定义笛卡尔空间直角坐标   
```js
var cartesian3 = new Cesium.Cartesian3(-2764128,4787610,3170373);`   
```

**屏幕坐标系**：  
是二维笛卡尔平面坐标系，原点为屏幕左上角，水平方向为x轴，垂直方向为y轴。Cesium中用Cartesian2定义屏幕坐标。
```js
var cartesian2 = new Cesium.Cartesian2(x,y);
```

**角度制转弧度制**

- 将经纬度从角度制转为弧度制，再通过Cartographic构造函数定义地理坐标。使用函数`Cesium.Math.toRadians(degrees)`配合`Cartographic`构造函数或`Cesium.Cartographic.fromRadians(longitude, latitude, height)`函数

    - 
    ```js
    var cartographic = new Cesium.Cartographic(Cesium.Math.toRadians(120),Cesium.Math.toRadians(30),1000)
    var cartographic = Cesium.Cartographic.fromRadians(Cesium.Math.toRadians(120),Cesium.Math.toRadians(30),1000)
    ```
- 通过Cartographic类中的`fromDegrees`函数定义
    - 
    ```js
    var cartographic = Cesium.Cartographic.fromDegree(120,120,1000)
    ```

**弧度制转角度制**

- 使用`Cesium.Math.toDegrees(radians)`

**弧度制坐标转三维笛卡尔坐标**  

- 使用函数`Cesium.Cartesian3.fromRadians(longitude, latitude, height)`
    - 
    ```js
    var cartesian3 = Cesium.Cartesian3.fromRadians(2.09,0.52,2000);
    ```
- 使用函数`cartographicToCartesian(cartographic,result)`
    - 
    ```js
    var cartesian3 = Cesium.Ellipsoid.WGS84.cartographicToCartesian(cartographic);
    ```

**三维笛卡尔转弧度制**  

- 使用函数`Cesium.Cartographic.fromCartesian(cartesian)`
    - 
    ```js
    var cartesian2 = new Cesium.Cartesian3(-2749573,4811111,3151552);
    var cartographic = Cesium.Cartographic.fromCartesian(cartesian3);
    ```
- 使用函数`cartesianToCartographic(cartesian)`
    - 
    ```js
    var cartographic = Cesium.Ellipsoid.WGS84.cartesianToCartographic(cartesian3);
    ```

**角度制转三维笛卡尔**  

- 使用函数`Cesium.Cartesian3.fromDegrees(longitude, latitude, height)`
    - 
    ```js
    var cartesian3 = Cesium.Cartesian3.fromDegrees(120,30,1000);
    ```
- 先转为弧度制再转为三维笛卡尔坐标


**三维笛卡尔转角度制**  

- 先转为弧度制坐标，再转为角度制


| 转换方向                      | 常用方法                                                |
| ------------------------- | --------------------------------------------------- |
| 角度制经纬度 → 弧度制地理坐标          | `Cesium.Cartographic.fromDegrees(lon, lat, height)` |
| 角度 → 弧度                   | `Cesium.Math.toRadians(degrees)`                    |
| 弧度 → 角度                   | `Cesium.Math.toDegrees(radians)`                    |
| 弧度制地理坐标 → Cartesian3      | `Cesium.Cartesian3.fromRadians(lon, lat, height)`   |
| Cartesian3 → Cartographic | `Cesium.Cartographic.fromCartesian(cartesian)`      |
| 角度制经纬度 → Cartesian3       | `Cesium.Cartesian3.fromDegrees(lon, lat, height)`   |


## Cesium三维空间相机
相机的飞行定位方法主要基于`Camera、Viewer`这两个类实现  

### Camera类
**camera.flyTo(options)**:让相机从当前位置飞到目标位置，有飞行动画。  

??? example
    ```js
    viewer.camera.flyTo({
    destination: Cesium.Cartesian3.fromDegrees(120.1, 30.3, 1000),
    orientation: {
        heading: Cesium.Math.toRadians(0),
        pitch: Cesium.Math.toRadians(-30),
        roll: 0
    }
    })
    ```

- destination：相机要飞到哪里
- Rectangle：矩形范围，让相机飞到一个区域，而不是一个点，可以用 Rectangle
- orientation：相机朝哪个方向看 
    - 法一：heading & pitch & roll，偏航角（左右转头）&俯仰角（上下抬头低头）&翻滚角（歪头侧翻）
    - 法二：direction & up，相机看向哪个方向&相机的上方朝哪个方向

**camera.setView(options)**：直接设置相机位置，没有飞行动画  

??? example
    ```js
    viewer.setView({
        destination:Cesium.Cartesian3.fromDegrees(120.1,30.3,1000),
        orientation:{
            heading:Cesium.Math.toRadians(0),
            pitch: Cesium.Math.toRadians(-30),
            roll: 0
        }
    })
    ```

参数和`flyTo`差不多  

**looAt(target,offset)**:让相机看向某个目标点  

??? example
    ```js
    const target = Cesium.Cartesian3.fromDegrees(120.1, 30.3, 0)

    viewer.camera.lookAt(
    target,
    new Cesium.HeadingPitchRange(
        Cesium.Math.toRadians(0),
        Cesium.Math.toRadians(-30),
        1000
    )
    )
    ```

- target：目标位置  
    - HeadingPitchRange 中:
    - heading：从哪个方向看
    - pitch：以什么俯仰角看
    - range：距离目标多远
- offset：相机相对于目标的偏移  

### Viewer类的相机方法
**viewer.flyTo(target, options)**：飞到某个对象  

target可以是Entity、DataSource、ImageryLayer、Cesium3DTileset  

??? example
    创建一个点，然后让相机飞到这个点附近  
    ```js
    const entity = viewer.entities.add({
    position: Cesium.Cartesian3.fromDegrees(120.1, 30.3),
    point: {
        pixelSize: 10
    }
    })
    viewer.flyTo(entity)
    ```

**viewer.zommTo(target,offset)**:直接把视角调整到某个对象上    
`viewer.flyTo`有飞行动画，`viewer.zoomTo`没有飞行动画，直接定位到对象附近  

### 主要相机参数
- heading偏航角（弧度）
    - 绕负 Z 轴旋转，顺时针为正，默认为正北方向，值为 0。
    - heading 控制相机朝哪个水平方向看，左右转头
    - `heading: Cesium.Math.toRadians(0)`表示朝北
    - `heading: Cesium.Math.toRadians(90)`表示朝东
- pitch俯仰角
    - 绕负 Y 轴旋转，顺时针为正，默认为俯视，值为 -90。
    - pitch 控制相机是低头看还是抬头看。
    - `pitch: Cesium.Math.toRadians(-90)`表示垂直向下看（俯视）
    - `pitch:Cesium.Math.toRadians(-30)`表示斜着往下看
- roll翻滚角
    - 绕正 X 轴旋转，顺时针为正，默认为 0。
    - roll 控制相机画面是否倾斜旋转，类似飞机侧翻
    - 一般写`roll:0`

```js
orientation: {
  heading: Cesium.Math.toRadians(0),    // 朝向
  pitch: Cesium.Math.toRadians(-30),    // 俯仰
  roll: 0                               // 翻滚
}
```

### 获取当前相机参数
- `viewer.camera.heading/pitch/roll`用于获取三维场景中当前相机的偏航角、俯仰角和翻滚角
- `viewer.camera.positionWC`用于获取当前相机在世界坐标系中的坐标（单位：米）
- `viewer.camera.positionCartographic`用于获取相机当前的经纬度坐标（经度与纬度以弧度表示，高度单位为米）

??? example
    将月牙楼设为相机目标，并向正北、俯视 30° 观察。  
    ```js
    const target = Cesium.Cartesian3.fromDegrees(经度, 纬度, 高度)
    viewer.camera.lookAt(
    target,
    new Cesium.HeadingPitchRange(
        Cesium.Math.toRadians(0),
        Cesium.Math.toRadians(-30),
        500
    )
    )
    ```
    `HeadingPitchRange(heading, pitch, range)`
    ```
    heading = 0°，从正北方向看
    pitch = -30°，向下俯视 30°
    range = 500，距离目标 500 米
    ```


## Cesium事件处理方法
### 鼠标事件
**1.创建事件处理器**  
```js
//实例化ScreenSpaceEventHandler
const handler = new Cesium.ScreenSpaceEventHandler(viewer.canvas)   
```
`viewer.canvas` 是 Cesium 渲染三维场景的画布  

**2.注册事件监听**  
使用`ScreenSpaceEventHandler`类的`setInputAction(action, type, modifier)`方法为该实例化对象注册鼠标事件的监听。`action`为事件发生时执行的回调函数，该函数接收一个事件对象作为参数，事件对象通常包含与事件有关的位置信息；`type`为事件类型；`modifier`为键盘事件修饰符。

```js
handler.setInputAction(function(event) {
  console.log(event.position)
}, Cesium.ScreenSpaceEventType.LEFT_CLICK)
```
