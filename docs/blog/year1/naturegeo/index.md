## 感受
这门课对高中学过地理的同学十分友好，上课时发现大部分概念是熟悉的。  
三篇论文分别是西湖成因，河流与自然环境的相互影响，气候与物候（竺老那篇论文的读后感），除了内容写丰富之外，排版对拿高分来说十分重要。
期末加油背吧。。。虽然高中地理有基础，但是具体的定义，尤其是涉名词解释，还是要花时间记忆的，98上那几份圣遗物已经足够了。对了一定要多参考一下历年卷，填空题几乎全是原题。
