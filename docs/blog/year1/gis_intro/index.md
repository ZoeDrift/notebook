## 感受
地信导总体不难，作业也不多。lab的实验指导非常详细，实验报告只需要对着一步步做并截图就可以了，还有一次调研报告给的时间也比较长，我写了5k左右。  
考试是大部分的理论和少部分的实验描述，最后一节课会发一个重点的pdf，直接背就行了，但是最后一节课那个划的“重点中的重点”不用理会，最好还是把pdf完整背下来
